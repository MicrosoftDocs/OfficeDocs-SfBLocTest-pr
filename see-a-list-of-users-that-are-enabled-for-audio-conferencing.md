# Ver una lista de usuarios habilitados para conferencias de acceso telefónico local

Después de habilitar a los usuarios de su organización para las conferencias de acceso telefónico local, podrá ver la lista de los usuarios habilitados. En la lista también verá por cada usuario de la lista el tipo de proveedor de servicios de conferencias de acceso telefónico local que está usando, el número de teléfono de acceso telefónico predeterminado del usuario y, si su organización no está habilitada para los identificadores de conferencia dinámicos, los identificadores de conferencia estáticos de las reuniones de acceso telefónico local que se organicen.
  
    
    


## Ver una lista de los usuarios


1. Inicie sesión en Office 365 con su cuenta profesional o educativa.
    
  
2. Vaya a **Centro de administración de Office 365** > **Skype Empresarial**.
    
  
3. En el **Centro de administración de Skype Empresarial**, en el panel de navegación de la izquierda, vaya a **Conferencias de acceso telefónico local** y luego a **Usuarios de acceso telefónico**.
    
  

## ¿Qué más debo saber?


- Cuando ve la lista de usuarios que están habilitados, puede seleccionar un usuario de la lista y usar el panel de acciones para editar la configuración de conferencias de acceso telefónico local para ese usuario.
    
  
- Cuando se selecciona un solo usuario que está configurado para usar Microsoft como proveedor de servicios de conferencia de acceso telefónico local, puede ver el número de teléfono predeterminado y, si su organización no está habilitada para los identificadores de conferencia dinámicos, restablecer el identificador de conferencia para las reuniones que organice el usuario.
    
  
- Cuando selecciona un solo usuario configurado para usar un proveedor de servicios de conferencias de acceso telefónico local de terceros, puede ver el nombre del proveedor de servicios de conferencias de acceso telefónico local, el número de teléfono de pago y el número de teléfono gratuito (si se han configurado).
    
  
- Puede usar las opciones de filtro para mostrar los usuarios que tienen:
    
  - **Conferencias de acceso telefónico local activadas**
    
  
  - **Conferencias de acceso telefónico local desactivadas**
    
  
  - **Proveedor de conferencias - Microsoft**
    
  
  - **Proveedor de conferencias - Otros**
    
  
- Puede usar el botón Buscar para buscar un usuario individual en la lista.
    
  
- Puede seleccionar más de un usuario y hacer lo siguiente:
    
  - Seleccionar un número predeterminado diferente para estos usuarios.
    
  
  - Desactivar la conferencia de acceso telefónico para el usuario al establecer el proveedor en **Ninguno**.
    
  
  - Cambie a Microsoft como proveedor de conferencias de acceso telefónico local si al usuario se le ha asignado una licencia **Conferencias RTC de Skype Empresarial**.
    
  
  - Permitir o no permitir a los usuarios anónimos activar las reuniones telefónicas de los usuarios seleccionados.
